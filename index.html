整趟以你已訂的 15:15 羚羊谷 為時間軸往前往後排。大峽谷走「南口進、東口 Desert View 出」順路接 89 號公路到 Page，不走回頭路。先看下面三個必讀提醒——這趟最大的挑戰不是路線，是「一天 ~18 小時、深夜開車回程」的體力與安全。

⚠️ 必讀 ① — 這是 ~18 小時、開車 ~10 小時的硬行程 因為 15:15 的團 + 馬蹄灣 + 回程 4.5 小時是固定的，深夜（約 23:00）開車回到鳳凰城躲不掉。
→ 強烈建議：兩人輪流開、備咖啡因、夜間提防鹿/麋鹿橫越公路。若覺得太硬，認真考慮在 Page 過一晚再回。
⚠️ 必讀 ② — 15:15 正逢七月午後雷雨高峰，且當天來回無補位緩衝 羚羊谷遇暴洪威脅會臨時取消。當天早上務必查 Page 天氣、並先確認你訂的團「取消/退費政策」。萬一被取消，當天無法改期（要趕回鳳凰城）——備案：改去馬蹄灣 + 現場走 Lower Antelope 候補。
提醒 ③ — 15:15 在七月已過「光束時段」 經典光束約在 10:30–13:30。15:15 你會看到柔和發光的峽谷壁、但不會有天降光柱。若你要的是光束、且還能改訂，建議換中午前的團。

①當天時程表（以 15:15 羚羊谷為軸）

含大峽谷版本。所有時間為概估，沿途觀景會延後，務必抓緊以準時趕上 15:15。

04:30
鳳凰城出發北上（天未亮）要清晨出發 → 建議前一晚先把車取好，租車中心清晨可能未開或排隊。
~08:00
抵大峽谷南緣（南口進）Mather Point / Yavapai Point 觀景；夏季 9:30 前進場可避開排隊長龍。
08:00–11:30
大峽谷 + 沿 Desert View Drive 東行邊開邊停觀景點，往東口移動，不走回頭路。
~11:30
Desert View 瞭望塔（東口出）→ 接 89 號往 Page東口 → Page 約 2 小時。
~13:30
抵 Page：午餐 + 把油加滿Page 之後加油站稀疏，務必滿油。
~14:30
到羚羊谷報到處提早 30–45 分鐘報到（導覽團規定，遲到不退費）。
15:15
羚羊谷導覽團 已預訂全程約 1–1.5 小時，約 16:45 結束。
~17:00
馬蹄灣 Horseshoe Bend（離羚羊谷約 15 分）步行 1.2 mi 來回、全程曝曬；午後較涼、光線漸佳。停車場開到 19:30。
~18:15
啟程返回鳳凰城（~4.5 小時）想看日落可待到 ~19:30，但回程會更晚到。
~22:45
抵鳳凰城夜間長途駕駛，務必輪流開、勿疲勞硬撐。
想更安全/輕鬆？拿掉大峽谷 大峽谷只影響你的上午（回程深夜到家是 15:15 鎖定的、跟去不去大峽谷無關）。若拿掉大峽谷：睡飽、09:00 出發 → 13:30 到 Page，行程鬆很多，還省下 $250 非居民年票。回程一樣約 23:00 到。

②大峽谷實務（順遊重點）

  • 路線：南口（South Entrance）進、東口（Desert View）出最順路去 Page，不繞回頭。
  • 費用（台灣籍 = 非居民）：閘口逐人付 = $35/車 + $100×3 = $335；改買 $250 非居民年票（含全車最多 4 人、免加收）較划算。線上購買，存手機或列印。
  • 時間：夏季 9:30–16:00 進場可排隊達 2 小時，9:30 前抵達最順。
  • 停留約 3.5 小時就要往 Page 移動，否則趕不上 15:15。觀景重點：Mather Point、Desert View 瞭望塔。

③羚羊谷 & 馬蹄灣

羚羊谷 Antelope Canyon（15:15 已訂）

須隨原住民導覽團進入、不能自行進場。提早 30–45 分到報到處，遲到不退。防塵、廣角鏡；確認你訂的團對大背包/腳架的規定。導遊小費備現金。

馬蹄灣 Horseshoe Bend

離 Page 約 15 分。停車 $10/車（國家公園年票不通用）。步行 1.2 mi 來回、無遮蔭、觀景台無護欄。七月務必帶足水、注意安全。

⑤七月當天來回必備

安全 / 防暑（最重要）

  • 第二位可輪流開的駕駛（已登記）
  • 每人 2–3 公升以上飲水 + 電解質
  • 咖啡因（應付深夜回程）
  • 高係數防曬 + 帽子 + 太陽眼鏡
  • 車用遮陽板

行程 / 證件

  • 台灣駕照 + 國際駕照 + 護照 + 信用卡
  • 羚羊谷預訂確認 + 導遊小費現金
  • 大峽谷 $250 年票（若去）存手機
  • 離線地圖（沿途收訊弱）
  • 出發前查 Page 當日天氣（防暴洪取消）
  • 行動電源、車充
